# 📄 产品需求文档 (PRD)：【来财】(LaiCai) 量化投研辅助系统 v1.0

## 1. 产品愿景与定位
**“来财”** 是一款专为个人投资者打造的**防幻觉、强逻辑、极高数据安全性**的 AI 量化盯盘与投研辅助系统。
系统通过量化指标监控核心持仓与观察名单，在触发关键交易条件时，结合大模型（LLM）的宏观与基本面分析，向用户推送具有高信噪比的投资操作建议。
*   **当前阶段重点**：聚焦个人静态持仓与关注名单的深度监控与分析。
*   **绝对底线**：系统为**纯旁路辅助工具**，仅提供建议，**绝对禁止**包含任何自动下单或交易接口调用的逻辑。

## 2. 核心设计原则（业务红线）
编码频道在进行详细设计时，必须严格遵守以下业务底线：
1.  **数据真实性第一（零幻觉）**：数据获取必须是100%客观的物理世界数据，且必须经过交叉验证。绝不允许将空值或存疑数据交给 AI 去“推测”或“脑补”价格。
2.  **严格的权责分离（解耦）**：大模型（AI）仅具备“逻辑分析与撰写报告”的权限，**严禁**让大模型直接参与数学计算或数据爬取。
3.  **高扩展性预留**：V1.0 虽只针对几十只股票的名单，但底层数据获取与规则计算的接口设计，必须具备未来横向扩展至“全市场数千只股票同时扫描”的吞吐量潜力。

## 3. 功能模块需求说明 (Functional Requirements)

### 📌 模块 0：任务调度与编排模块 (Orchestrator)
*   **业务需求**：系统的中控大脑，负责按照预设的时间表唤醒系统，并按顺序串联后续三大模块。
*   **输入**：定时任务触发指令、目标股票池名单（配置文件）。
*   **输出**：完整的系统执行日志与控制流转。

### 📌 模块一：多市场行情数据中心 (Data Center) - 【高危防线】
*   **业务需求**：负责从外部获取标准化的行情数据，屏蔽各类数据源差异。提供 A 股、港股、美股数据。
*   **核心约束（必须严格实现）**：
    1.  **原值透传原则（禁止二次加工）**：必须直接提取数据源 API 提供的原生字段（如 API 返回的 `涨跌额`、`涨跌幅`）。**严禁**程序化使用“今日现价 - 昨日收盘价”自行计算涨跌额（以防止因除权除息、拆股等公司行为导致的巨大计算灾难）。
    2.  **双路交叉验证（熔断机制）**：获取任何证券的核心报价时，必须同时请求两路独立的数据源（例如：主路 AkShare，备路 Yahoo Finance 或 QVeris API）。仅当两路数据的价格偏差在极小合理阈值内时，数据才被标记为“有效”并放行下游。**一旦两路数据出现异常偏差或某一路超时，模块直接抛出致命错误（Fatal Error）并终止本次全流程**，绝不给后续模块提供制造幻觉的机会。
    3.  **周期支持**：支持**日线 (1D)、周线 (1W)、月线 (1M)**。明确放弃复杂的半月线支持。

### 📌 模块二：量化规则与指标计算引擎 (Quant Engine)
*   **业务需求**：基于“模块一”提供的经过双源验证的绝对安全数据，进行数学层面的指标计算和条件比对。
*   **计算能力要求**：
    *   **基础比对**：判断最新价是否落入用户预设的“买入/卖出区间”。
    *   **技术指标**：支持计算布林带 (BOLL)、MACD、RSI 等主流技术指标。
*   **输出要求**：输出**结构化的命中事件包**（例如：价格触达买入区间，且日线跌破布林带下轨）。未命中规则的股票直接静默过滤。

### 📌 模块三：AI 投研与报告生成大脑 (AI Analyst)
*   **业务需求**：接收“模块二”输出的冰冷警报，利用大模型结合真实世界的新闻，输出具有业务价值的操作建议研报。
*   **动作要求**：
    1.  **归因分析**：发生异动时，通过检索抓取该股票最新财经新闻、财报或宏观异动事件。
    2.  **建议生成**：结合用户“适中偏谨慎、长线投资”的风险偏好，对事件进行解读，并给出明确的操作建议。
*   **输出要求**：向 Discord 频道发送排版清晰、带有精确双源数据时间戳的最终研报。

## 4. 核心业务流程描述 (User Journey)
1. **触发**：到达设定时间，编排模块启动。
2. **取数与验真**：数据中心并行调用双路数据源，校验两者一致性。若不一致，报警并退出；若一致，提取**原生字段**组装成标准数据帧。
3. **计算**：量化引擎对验真后的数据进行比对，发现 `股票A` 跌入买点且 RSI 超卖。
4. **研判**：AI 大脑接收 `股票A` 事件，联网查阅资讯，进行逻辑推理。
5. **通知**：AI 大脑在 Discord 推送警报研报。
6. **终止**：系统结束运行。
